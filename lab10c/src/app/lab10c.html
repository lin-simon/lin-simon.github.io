Welcome to my LAB10c page built using ANGULAR JS

Hello, my name is Simon.

This page was built with the help of the Angular JS framework and I decided to host it on my personal portfolio website.

I chose to learn Angular since it is a TypeScript language and I am already quite familiar with JavaScript, so it was an easy transition. The installation required me to install node.js as an environment and package manager so that I could run command line arguments to set up my workspace. Angular was quickly installed by running the following argument in my terminal:

C:\My Files\VSProjects\lin-simon.github.io> npm install -g @angular/cli

The difficult part of building the page was setting up the virtual environment for node.js. There are a lot of dependencies for the framework.

Other than that, hosting was relatively easy since I already have hosted using github pages in the past.

This page was built with Angular's Component styles which encapsulate content to specific pages, which was initially very syntactically confusing to understand.
